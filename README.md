# Voice-control-ai-assistant
A smart Voice Control Assistant built using Python that performs tasks like playing music, opening websites, sending messages, and answering queries using AI.

## 🚀 Features
- 🎙️ Voice command recognition
- 🕒 Tells current time
- 🌐 Opens websites (Google, YouTube)
- 🎵 Plays songs from YouTube
- 💬 Sends WhatsApp messages
- 📖 Fetches information from Wikipedia
- ➗ Performs basic calculations
- 🤖 AI-based responses using OpenAI API

## 🛠️ Technologies Used
- Python
- SpeechRecognition
- Pyttsx3
- PyWhatKit
- Wikipedia
- OpenAI API
